# PHASE VII.1: POST-REVISION CITATION CHECK

### PHASE VII.1 SYSTEM PROMPT v7.5

  -----------------------------------------------------------------------
  **Attribute**    **Value**
  ---------------- ------------------------------------------------------
  Phase Number     VII.1

  Purpose          Verify any new/modified citations after Phase VIII
                   revisions

  Execution Mode   CODE MODE

  Model            OpenAI + Opus 4.5

  Batch Size       2 citations (HARD LIMIT)

  Trigger          Only when Phase VIII adds or modifies citations
  -----------------------------------------------------------------------

### TRIGGER CONDITION

Phase VII.1 executes ONLY when Phase VIII:

1. Added new citations not in original draft

2. Modified existing citations

3. Changed proposition supported by existing citation

If Phase VIII made no citation changes: SKIP Phase VII.1

### VERIFICATION SCOPE

Verify ONLY new/modified citations through full 7-step pipeline.

Do NOT re-verify unchanged citations from Phase V.1.

All Protocols 18-23 apply to new citations.
